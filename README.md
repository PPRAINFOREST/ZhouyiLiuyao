# 周易算卦 Agent

基于揲蓍布卦法的周易占卜智能助手，使用 Go 语言、Eino 框架和豆包大模型。

## 特性

- 🎯 **传统揲蓍布卦法**：完整实现周易三变成一爻、六爻成一卦的传统占卜方法
- 💡 **AI 智能解卦**：集成豆包大模型，提供专业的卦象解读
- 🔮 **正态分布随机数**：使用 Box-Muller 变换生成正态分布，拒绝采样确保范围
- 📊 **五行分析**：完整的八卦五行统计，推荐最佳农历月份
- 🌱 **互卦分析**：分析事物发展的潜在趋势
- ✨ **流式输出**：逐字显示解卦结果，增加仪式感
- 🤖 **智能问题总结**：AI 自动将问题总结为简洁短语

## 快速开始

### 前置要求

- Go 1.26.1+
- 火山引擎方舟（豆包）API Key

### 安装步骤

1. **克隆项目**

```bash
git clone <your-repo-url>
cd zhouyi-divination
```

2. **配置环境变量**

复制环境变量模板：

```bash
cp .env.example .env
```

编辑 `.env` 文件，填入您的火山引擎方舟 API Key：

```env
ARK_API_KEY=your_api_key_here
ARK_MODEL_ENDPOINT=your_model_endpoint_here
```

3. **安装依赖**

```bash
go mod download
```

4. **运行程序**

```bash
go run .
```

## 使用说明

### 基本操作

1. **启动程序**

```bash
go run .
```

2. **进行占卜**

在命令行输入您的问题，加上算卦关键词即可：

```
请输入您的问题或指令: 帮我算一算，我想投资买房怎么样
```

支持的算卦关键词：`算卦`、`起卦`、`占卜`、`卜卦`、`揲蓍`、`布卦`、`算一下`、`算一算`

3. **退出程序**

```
请输入您的问题或指令: exit
```

### 占卜流程

1. **问题理解**：AI 自动将您的问题总结为 2-6 个汉字的简洁短语
2. **揲蓍布卦**：传统三变成一爻，六爻成一卦
3. **卦象展示**：完整显示本卦、变爻、互卦、五行分析
4. **AI 解卦**：大模型根据卦象提供专业解读

## 核心功能

### 揲蓍布卦算法

完整实现周易传统揲蓍布卦法：

- **三变成一爻**：每爻经过三次分二、挂一、揲四、归奇
- **六爻成一卦**：从下往上依次生成初爻至上爻
- **变爻识别**：老阳（九）、老阴（六）为变爻，少阳（七）、少阴（八）不变

### 五行分析

- **八卦五行对应**：乾兑金、震巽木、坎水、离火、坤艮土
- **详细统计**：分别统计每个八卦的数量
- **最佳月份**：根据五行旺衰推荐最佳农历月份

### 解卦规则

根据变爻数量采用不同的解卦策略：

| 变爻数 | 解卦方法 |
|--------|----------|
| 0 | 仅参考本卦卦辞 |
| 1 | 参考本卦卦辞 + 变爻爻辞 |
| 2 | 本卦卦辞 + 第一变爻爻辞（主）+ 第二变爻爻辞（次） |
| 3 | 本卦卦辞 + 变卦卦辞 |
| 4 | 变卦卦辞 + 第一不变爻爻辞（主）+ 第二不变爻爻辞（次） |
| 5 | 变卦卦辞 + 不变爻爻辞 |
| 6 | 仅参考变卦卦辞 |
