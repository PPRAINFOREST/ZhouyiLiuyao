# 快速开始指南

## 5 分钟快速上手

### 1. 准备工作

确保你已经安装：
- Go 1.26.1 或更高版本
- 一个火山引擎方舟账号（用于获取 API Key）

### 2. 获取 API Key

1. 访问 [火山引擎方舟控制台](https://console.volcengine.com/ark)
2. 注册/登录账号
3. 创建一个应用并获取 API Key
4. 记下你的 API Key

### 3. 配置项目

```bash
# 进入项目目录
cd zhouyi-divination

# 复制环境变量模板
cp .env.example .env

# 编辑 .env 文件，填入你的 API Key
# 打开 .env 文件，找到这一行：
# ARK_API_KEY=your_api_key_here
# 修改为：
# ARK_API_KEY=sk-xxxxxxxxxxxxx
```

### 4. 安装依赖

```bash
go mod download
```

### 5. 运行程序

```bash
go run .
```

看到这个界面就说明成功了！

```
========================================
         周易算卦 Agent (揲蓍布卦法)
========================================

使用说明：
  1. 输入您的问题或事项
  2. 输入 '算卦'、'起卦'、'占卜' 等关键词进行算卦
  3. 输入 'exit' 或 'quit' 退出程序

请输入您的问题或指令: 
```

### 6. 开始占卜

试试输入：

```
请输入您的问题或指令: 算一算我最近的财运怎么样
```

然后程序会：
1. 理解你的问题
2. 进行揲蓍布卦
3. 显示卦象
4. AI 为你解卦

### 7. 退出程序

```
请输入您的问题或指令: exit
```

## 常见问题

### Q: 提示找不到 go 命令？

A: 请先安装 Go，访问 https://go.dev/dl/ 下载安装

### Q: 提示 API Key 错误？

A: 请检查 `.env` 文件中的 API Key 是否正确，不要有多余的空格

### Q: 程序卡住不动？

A: 请检查网络连接，确保能访问火山引擎方舟服务

### Q: 如何只运行不编译？

A: 使用 `go run .` 即可，不需要编译

### Q: 如何编译成可执行文件？

A: 
```bash
# 编译
go build -o zhouyi .

# 运行
./zhouyi
```
